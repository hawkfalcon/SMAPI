PatchImage(Texture2D, Nullable<Rectangle>, Nullable<Rectangle>, PatchMode)

Overwrite part of the image.

Declaration
void PatchImage(Texture2D source, Rectangle? sourceArea = default(Rectangle? ), Rectangle? targetArea = default(Rectangle? ), PatchMode patchMode = PatchMode.Replace)
Parameters
Type Name Description
Microsoft.Xna.Framework.Graphics.Texture2D source

The image to patch into the content.
System.Nullable<Microsoft.Xna.Framework.Rectangle> sourceArea

The part of the source to copy (or null to take the whole texture). This must be within the bounds of the source texture.
System.Nullable<Microsoft.Xna.Framework.Rectangle> targetArea

The part of the content to patch (or null to patch the whole texture). The original content within this area will be erased. This must be within the bounds of the existing spritesheet.
PatchMode patchMode

Indicates how an image should be patched.
Exceptions
Type Condition
System.ArgumentNullException

One of the arguments is null.
System.ArgumentOutOfRangeException

The targetArea is outside the bounds of the spritesheet.
System.InvalidOperationException

The content being read isn't an image.
